Flu Course Part 2: Analyzing and Visualizing Data


In this part of the workshop we'll learn basic summary statistics and visualization with SQL, hypothesis testing with SQL, Pandas and scipy, and linear modeling with SQL and scikit-learn. Since we gathered up all our data in the last part of the workshop, we'll use some questions to guide our data investigation.

  • Does Ethnicity Impact Vaccination Rates?

    Learn basic summary statistics, t-test and single-factor modeling, and visualization with SQL
  • Do vaccination rates impact flu rates?

    Learn analytical SQL and linear modeling without interactions
  • Does GDP Explain Flu Rates?

    Learn hypothesis testing with SQL, Pandas and Scipy
  • Does Living in Cities Inflence Flu Rates?

    Learn two-factor linear modeling with interaction effects
  • How Do Sanitation and Clean Water Impact Flu?

    Learn and logistic regression with SQL and Python
