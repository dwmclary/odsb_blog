Big Data Science Bootcamp



How to use Data Science to impress your friends...

Hey there, you’ve reached the blog portion Big Data Science Bootcamp! You can find our series of corresponding lectures on Youtube here.

The goal of Bootcamp is to explain how to tell meaningful stories with data using best-of-breed tools. We’ll be using popular developer tools to treat tangible data problems from ingest to insight. No contrived examples here, our courses use real (publically available) data sets to teach skills to deal with data ingestion, quality, analysis, and visualization.

Get started with one of our courses below! If you have questions along the way, be sure to check out our Command Shortcuts Index and FAQ’s.
