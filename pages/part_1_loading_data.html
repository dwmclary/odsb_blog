Flu Course Part 1: Loading Data


In this part of the workshop we'll learn how to load different types of text and web data into a database using Python and other tools.

    Learn to load and share CSV data
  • World Health Organization Country-Level Flu Data

  • HHS Flu Vaccination Data

    Learn to load JSON data
  • OIE Pathogenic Flu Data

    Learn how to scrape embedded data from the web
  • Collecting Web Data With Pandas

    Learn how to search for useful data from around the world
