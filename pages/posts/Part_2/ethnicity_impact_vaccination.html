Does Ethnicity Impact Vaccination Rates?¶

We know flu shots are important, but who gets them? Is there a difference in vaccination rates between ethnic groups in the US?

Fortunately, we can find this out using hypothesis testing. First, we'll need to hypothesize a bit. The second question in the preceding paragraph is close to a hypothesis, but it's not quite specific enough. Let's rephrase it such that we can test it:

H1: There is a difference in mean vaccination rate between different ethnic groups in the US.

That's a hypothesis we can use, but in order to test it, we need more. Hypothesis tests use data to determine whether one can accept or reject a null hypothesis. H1 is what we'll call the alternative hypothesis. The null hypothesis will be less exciting:

H0: The difference in mean vaccination rate between different ethnic groups in the US is random.

Our test will allow us to determine whether or not H0 explains the data. If it does, then we can conclude there is no difference. If it does not, then there is a possibility that H1 explains the data. The is the key concept in hypthoesis testing: we cannot show that H1 explains the data, we can only show that the data is not random.

Now we're ready to test our groups. We're going to be using Student's t-test, which provides a hypothesis test on mean values that lines up exactly with our H0 and H1 hypothesis.

The t-test only compares 2 groups at a time, so we'll write a method which compairs each pair of groups. We'll apply this to the groups and place the results in a DataFrame for easy viewing.

In [12]: 
groups = [e for e in vac_rates_grouped_by_eth]

def pairwise_t_test(groups, sig=0.05):
    for i in range(len(groups)):
        grp = groups[i]
        for j in range(i,len(groups)):
            other = groups[j]
            if grp[0] != other[0]:
                r = ttest_ind(grp[1], other[1])
                reject = False
                if r[1] < sig:
                    reject=True
                yield grp[0], other[0], r[0], r[1], reject

t_test_results = pd.DataFrame(pairwise_t_test(groups), columns=["Group 1", "Group 2", "t stat.", "p-value", "reject H0"])

Looking at the results, we can see that there is a statitically significant difference between most of the groups. However, Hispanic and African-American populations show no statistical difference in their mean vaccination rate. This is a good example of how simple hypothesis testing can open up new questions. If the two populations have the same mean vaccination rate, is there a common factor we haven't considered which makes them more alike than the other groups?

Summary¶

In this excercise, we've covered what may be the most fundamental tool in a scientist's kit: the hypothesis test. While there are many kinds of hypothesis test, the t-test is among the most broadly used. To do this we:

  • Used SQL to extract data into Pandas
  • Plotted groupwise histograms to understand distribution shape
  • Used a t-test to examine pairwise differences in mean.

When considering how to analyze different populations or models, check to see if there's a hypothesis test which can help validate your thinking.
