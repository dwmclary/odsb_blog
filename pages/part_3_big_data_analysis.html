Flu Course Part 3: Analyzing Big Data with Apache Spark


In this part of the workshop we'll learn how to use tools from Apache Spark to derive insight from big data. We'll work with text from the news to show how we can use data analysis to determine what news articles are about, and then we'll draw conclusions about the flu from those analyses.

  • Basic Big Data

    with PySpark
  • Moving and Clustering Data

    with Sqoop and Spark
  • Finding Important Words

    with Spark
  • Trend Search

    with Big Data and SQL
  • Clustering the News

    with Spark and MLlib
  • Streaming News Analysis

    with Spark Streaming

Big Data Science Bootcamp was created by Dan McClary and Althea Vail Wallop
